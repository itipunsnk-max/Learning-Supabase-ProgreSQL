# Security Policy

## แนวทางรายงานปัญหา

หากพบ secret หลุด, ตัวอย่างที่เปิดสิทธิ์กว้างเกินไป หรือช่องโหว่ในตัวอย่างโค้ด ให้หยุดเผยแพร่รายละเอียดสาธารณะและแจ้งผู้ดูแล repository โดยตรง

## กฎสำคัญ

- `service_role` key ใช้เฉพาะ server-side และห้ามฝังใน Power Apps หรือ browser
- ใช้ environment variables สำหรับค่าการเชื่อมต่อ
- เปิด Row Level Security (RLS) ก่อนนำตารางไปใช้งานจริง
- ใช้ข้อมูลจำลองแทนข้อมูลผู้ใช้งานจริงในบทเรียน
