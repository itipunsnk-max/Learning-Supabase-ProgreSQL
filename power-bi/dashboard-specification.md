# Dashboard Specification

| Audience | Visuals | Decision |
| --- | --- | --- |
| Technician | assigned work, aging, status | จัดลำดับงานวันนี้ |
| Supervisor | backlog, SLA, workload | กระจายงานและเร่งงานเสี่ยง |
| Manager | MTTR, cost/site, repeat failure | วางแผนงบและ reliability |
| Executive | trend, downtime, critical assets | ตัดสินใจลงทุนและความเสี่ยง |
