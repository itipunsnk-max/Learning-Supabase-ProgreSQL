# API Assets

ตัวอย่าง REST request อยู่ที่ `api/examples/tickets.http`, JSON schema อยู่ที่ `api/schemas/ticket.json` และ Postman collection อยู่ที่ `api/postman/cmms.postman_collection.json`

ทุกตัวอย่างใช้ user access token และ anon key ที่ถูกป้องกันด้วย RLS; service role key ไม่ควรส่งไปยัง client
