## เกณฑ์ผ่านหลักสูตร

ผู้เรียนควรส่ง Lab ทุกสัปดาห์, ผ่าน quiz อย่างน้อย 70%, แสดงการทดสอบ RLS และนำเสนอ Final Project ที่มี architecture, database, security, API, dashboard และคู่มือผู้ใช้
