# Capstone Assessment

- [ ] Architecture Review: data flow และ security boundary อธิบายได้
- [ ] Database Review: keys, constraints, indexes, views และ migration ผ่าน
- [ ] Security Review: RLS matrix, secret management และ storage policy ผ่าน
- [ ] Application Demo: create/assign/repair/verify/close ครบ
- [ ] API Test: success, 4xx, 401, retry และ pagination
- [ ] Power BI Dashboard: KPI และ drillthrough ตรงกับข้อมูล
- [ ] Documentation Review: user/admin/troubleshooting manuals ครบ
- [ ] Production Readiness: backup, restore, monitoring, rollback และ support owner
