# Week 1 Assessment

## Knowledge Check / Quiz 10 ข้อ

1. CMMS ช่วยแก้ปัญหาใดในงานซ่อมบำรุง
2. Master Data และ Transaction ต่างกันอย่างไร
3. เพราะเหตุใด Asset ควรใช้ Lookup
4. Choice column เหมาะกับ field ใด
5. Delegation warning ส่งผลต่อผลลัพธ์อย่างไร
6. `SubmitForm` เหมาะกับกรณีใด
7. ทำไมการสร้าง Ticket Number ด้วยเวลาปัจจุบันเพียงอย่างเดียวไม่พอ
8. Critical ticket ควรมี workflow อย่างไร
9. การเก็บ location ต้องคำนึงถึงเรื่องใด
10. เหตุใด Development และ Production ต้องแยกกัน
