# Glossary ไทย–อังกฤษ

| Term | ความหมาย |
| --- | --- |
| API | ช่องทางให้ระบบแลกเปลี่ยนข้อมูล |
| REST | รูปแบบ API ที่ใช้ resource และ HTTP methods |
| JSON | รูปแบบข้อมูล text สำหรับ request/response |
| Database | แหล่งเก็บข้อมูลที่มีโครงสร้าง |
| Primary Key | ค่าระบุแถวแบบไม่ซ้ำ |
| Foreign Key | ความสัมพันธ์ไปยังตารางอื่น |
| Constraint | กฎบังคับความถูกต้องของข้อมูล |
| Index | โครงสร้างช่วยค้นหาเร็วขึ้น |
| View | query ที่นำเสนอเป็นตารางเสมือน |
| Trigger | การทำงานอัตโนมัติเมื่อเกิด database event |
| Function | logic ที่เรียกใช้ซ้ำได้ใน database/API |
| Authentication | การยืนยันตัวตน |
| Authorization | การตรวจสิทธิ์การเข้าถึง |
| JWT | token ที่บรรจุ claims ของ session |
| RLS | policy ควบคุมระดับแถว |
| Storage Bucket | พื้นที่เก็บไฟล์ |
| Signed URL | URL ชั่วคราวสำหรับไฟล์ private |
| SLA | ข้อตกลงเวลาการตอบสนอง/แก้ไข |
| MTTR | เวลาเฉลี่ยในการซ่อมเสร็จ |
| MTBF | เวลาเฉลี่ยระหว่างการขัดข้อง |
| Work Order | ใบสั่งงานซ่อม |
| Asset | ทรัพย์สิน/อุปกรณ์ที่ติดตาม |
| Audit Log | ประวัติการกระทำที่ตรวจสอบย้อนหลังได้ |
| Soft Delete | ทำเครื่องหมายลบโดยไม่ลบแถวจริง |
| Pagination | แบ่งผลลัพธ์เป็นหน้า |
