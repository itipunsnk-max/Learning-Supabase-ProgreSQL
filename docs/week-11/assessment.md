# Week 11 Assessment

## Knowledge Check / Quiz 10 ข้อ

1. Unit, integration และ UAT ต่างกันอย่างไร
2. Test case ที่ตรวจสอบย้อนกลับได้ต้องมีอะไร
3. ทำไมต้องใช้ test data ที่ masked/synthetic
4. RLS ควรทดสอบผ่านช่องทางใด
5. Severity และ Priority ต่างกันอย่างไร
6. UAT sign-off ควรมาจากใคร
7. Regression test สำคัญหลังแก้ policy เพราะอะไร
8. Incident runbook ควรมีลำดับใด
9. Known defect ควรบันทึกอะไร
10. Release gate ควร block เมื่อใด
